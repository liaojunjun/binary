TextDecoder

将字节转为字符串

const decoder = new TextDecoder();
const uint8Array = new Uint8Array([
  228,
  189,
  160,
  229,
  165,
  189,
  229,
  145,
  128,
]);

const str = decoder.decode(uint8Array);

alert(str);

TextEncoder

将字符串转为字节

const encoder = new TextEncoder();
const uint8Array = encoder.encode("你好呀");

alert(uint8Array);
